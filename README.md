# Moon wars

HTML5 based tank game. Requires Firefox, Chrome or Opera.

## Features:
* 8 tanks and 2 air units
* 4 maps
* single player
* multi player
* open source
* full screen support
* using html5, no flash

## Getting Started
* http://viliusle.github.com/vl-tanks/
* or download and open index.html in modern browser to play dev version
